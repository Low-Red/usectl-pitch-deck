// 03 - Solution

One CLI. Your agent runs the whole stack.

usectl is a fully managed deployment platform built on serious Kubernetes infrastructure, giving you the control and flexibility of self-hosting without the operational overhead. CLI, MCP server, and dashboard are all peers - your AI agent deploys, provisions databases, manages domains, and reads logs without a human in the loop.

usectl machines create / deploy
Push a repo, agent picks the runtime, ships the code. Blue-green rollout, instant rollback.
usectl addons add postgres / redis / s3
Managed Postgres, Redis, MongoDB, NATS, S3-compatible storage - provisioned in seconds.
usectl envs set / vars set
Runtime and build-time config, per-app overrides. Secrets managed, not scattered.
usectl logs follow / status
Read live logs, diagnose issues, heal at runtime. The agent debugs production.
usectl domains add / verify
Custom domains, DNS, Cloudflare SSL - auto-renewed, zero config.
usectl machines pr-preview
Every PR gets its own live URL with isolated namespace. Auto-torn-down on merge.
Your agent doesn't just deploy. It runs the whole stack. You get the week back - not just the deploy hour.
